# Test 

## Requisitos

Asegúrate de tener instalado lo siguiente:

- [Node.js](https://nodejs.org/)

## Instalación

1. Clona el repositorio a tu máquina local:

   ```bash
   git clone https://github.com/Brunocarda2005/test.git
   cd test
   ```
2. Instala las depencencias:
   ```
   npm install
   ```

3. Correlo:
   ```
   npm run dev
   ```
